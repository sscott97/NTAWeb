📝 Method

NT50 and NT90 titres were calculated by linear interpolation on each replicate column independently. For each replicate, the luminescence value at 50% (or 90%) of the no-serum control (NSC) was identified, and the corresponding dilution was estimated by linear interpolation between the two flanking data points. The three per-replicate titres were then averaged to produce a single NT50 or NT90 per quadrant.

Where a sample's luminescence did not cross the threshold within the measured dilution range, a boundary value is reported: ≤ (the lowest dilution) if luminescence remained below the threshold at all dilutions, or ≥ (the highest dilution) if luminescence never fell below it.
