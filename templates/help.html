Help & Instructions

Welcome to the Neutralising Titre Automator information section!

This application takes the raw comma-separated value (CSV) data produced by the Kaleido software on the Perkin-Elmer Luminometer and processes it using an Excel template to automatically calculate neutralising antibody titres.

How to Use This Application
  1. Input CSV File: Upload raw data in CSV format (must be 8x12 blocks, Data Only mode).
  2. Template File: A default Excel template will be loaded automatically.
  3. Output File: The output file will be saved and downloadable after processing.
  4. Optional Assay Data: Enter pseudotype names and sample IDs, separated by commas.
  5. Process File: Processes your CSV and produces an Excel file with assay metadata.
  6. Create Graphs: After processing, choose graph colors and export dilution plots.
Additional Notes
  • CSV files must be saved in Data Only mode. Files in Standard mode will not be processed.
  • Template files should be stored in the application folder under a Templates/ subfolder.
  • Settings and presets (e.g. colors) can be saved via the Settings page.
